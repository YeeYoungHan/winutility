# 윈도우 tail 프로그램

### 소개

* 윈도우 도스창에서 실행할 수 있는 tail 프로그램 

### 라이선스

* 본 프로젝트는 GPLv3 라이선스입니다. 

### 사용법

```
dos> tail {-f filename} 
          {-g search word} 
          {-h} : show help message
          {-v} : show program version
          {-t} : search file from start position
          {-u} : Converted to utf8 string and output
```
