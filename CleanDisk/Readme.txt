디스크 안전 삭제 프로그램

1. 소개

 * 디스크의 남은 공간만큼 파일을 생성한 후, 삭제하는 프로그램
 * 디스크가 빈 상태에서 실행하면 디스크에 저장된 모든 데이터를 삭제할 수 있다.
 * OS 에서 파일 삭제시 실제 파일을 삭제하지 않고 파일 정보만 삭제하지 않고 HDD 빠른 포맷도 디스크의 row 데이터를 모두 삭제하지 않으므로 디스크의 row 데이터를 모두 삭제하기 위해서 개발한 프로그램

2. 라이선스

 * 본 프로젝트는 GPLv3 라이선스입니다. 

3. 사용법

 * 프로그램 사용법은 다음과 같습니다. 

 # CleanDisk.exe c:\
