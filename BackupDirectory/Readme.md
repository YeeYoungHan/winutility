# 백업 프로그램

### 소개

* 윈도우 도스창에서 편리하게 사용할 수 있는 백업 프로그램 개발 
* 윈도우 배치파일로 실행하면 N 개의 폴더에 대해서 백업이 가능하다.

### 라이선스

* 본 프로젝트는 GPLv3 라이선스입니다. 

### 사용법

* 프로그램 사용법은 다음과 같습니다. 

```
# BackupDirectory {원본 폴더 경로} {백업 폴더 경로} 
# BackupDirectory c:\Project d:\Project 
```

* N 개의 폴더를 편리하게 백업하는 방법은 다음과 같습니다. 
  * 백업을 위한 배치 파일을 생성한다. 
  * 생성된 배치 파일에 위의 예제와 유사하게 N개 입력한다. 
  * 배치 파일을 실행하여서 N 개의 폴더를 백업한다. 

```
[주의] 현재 버전에서는 폴더 경로 뒤에 \ 를 입력하면 제대로 동작하지 않을 수도 있습니다. 
```
