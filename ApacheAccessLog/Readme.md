# 아파치 access log 통계 프로그램

### 소개

* 윈도우 도스창에서 편리하게 사용할 수 있는 아파치 접속 통계 프로그램 개발
* 현재는 월별 트래픽 통계만 제공한다.

### 라이선스

* 본 프로젝트는 GPLv3 라이선스입니다. 

### 사용법

* 프로그램 사용법은 다음과 같습니다. 

```
# ApacheAccessLog.exe {apache access log folder} {통계 출력 폴더} 
```

* 위와 같이 실행하면 통계 출력 폴더에 아래의 파일들이 출력된다.
  * month_size.csv : 월별 트래픽 통계
