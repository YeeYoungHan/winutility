# 소스 코그 라인 개수 출력하는 프로그램

### 소개

* 소스 코그 라인 개수 출력하는 프로그램

### 라이선스

* 본 프로젝트는 GPLv3 라이선스입니다. 

### 사용법

* Folder 를 선택하면 해당 폴더에 포함된 모든 폴더의 소스 코드 라인 개수를 출력한다.
* 소스 코드로 인식하는 파일 확장자는 다음과 같다.

'''
"cpp", "c", "hpp", "h", "m", "mm", "cc", "java", "js", "kt", "cs", "html", "htm", "css", "php"
'''
