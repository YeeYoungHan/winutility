# BOM 헤더가 포함된 파일을 BOM 헤더가 제거된 파일로 저장하는 프로그램

### 소개

* 윈도우 도스창에서 BOM 헤더가 포함된 파일을 BOM 헤더가 제거된 파일로 저장하는 프로그램 

### 라이선스

* 본 프로젝트는 GPLv3 라이선스입니다. 

### 사용법

```
dos> RemoveBOM.exe {BOM filename} {output filename}

지정된 폴더에 포함된 모든 파일에서 BOM 헤더를 제거하는 방법은 다음과 같습니다.

dos> RemoveBOM.exe -d {folder full path}
```
